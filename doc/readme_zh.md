# NVDA YouTube音乐播放器

🎵 可访问的NVDA插件，用于搜索和播放YouTube和YouTube Music的音乐。

## 功能

- 🔍 搜索YouTube和YouTube Music
- 🎧 带NVDA反馈的无障碍音频播放器
- 🎚️ 音量、速度和音调控制
- 🔁 重复和自动播放模式
- 📻 Radio Mix发现新音乐
- 🔄 自动更新检查
- 🌐 支持11种语言

## 安装

1. 下载最新的`.nvda-addon`文件
2. 双击安装
3. 重启NVDA
4. 按`NVDA+Alt+Y`打开

## 键盘快捷键

| 按键 | 操作 |
|------|------|
| `NVDA+Alt+Y` | 打开播放器 |
| `↑` / `↓` | 音量 |
| `←` / `→` | 前进/后退1秒 |
| `空格` | 播放/暂停 |
| `Escape` | 关闭播放器 |
