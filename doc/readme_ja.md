# NVDA用 YouTube Music Player

🎵 YouTubeとYouTube Musicから音楽を検索・再生できるNVDAアドオン。

## 機能

- 🔍 YouTubeとYouTube Musicで検索
- 🎧 NVDAフィードバック付きのアクセシブルなプレイヤー
- 🎚️ 音量、速度、音程の調整
- 🔁 リピートと自動再生モード
- 📻 新しい音楽を発見するRadio Mix
- 🔄 自動アップデートチェック
- 🌐 11言語対応

## インストール

1. 最新の`.nvda-addon`ファイルをダウンロード
2. ダブルクリックでインストール
3. NVDAを再起動
4. `NVDA+Alt+Y`を押して開く

## キーボードショートカット

| キー | 動作 |
|------|------|
| `NVDA+Alt+Y` | プレイヤーを開く |
| `↑` / `↓` | 音量 |
| `←` / `→` | 1秒進む/戻る |
| `スペース` | 再生/一時停止 |
| `Escape` | プレイヤーを閉じる |
